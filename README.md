# EliteX HyperOS Enhancement

An all-in-one collection of systemless enhancement modules designed for **HyperOS 2 (Android 15)** and **HyperOS 3 (Android 16)**. Built for **Xiaomi, Redmi, and POCO devices**, EliteX HyperOS Enhancement combines premium visual improvements, hidden system features, performance optimizations, and quality-of-life enhancements into a single package.

Instead of installing multiple individual modules, EliteX brings together the most useful HyperOS modifications in one lightweight, easy-to-install solution. Whether you're running HyperOS 2 or HyperOS 3, this project aims to provide a smoother, cleaner, and more feature-rich experience while maintaining excellent system stability.

# 🚀 Installation

## Step 1

Download the correct module version for your HyperOS version.

- **HyperOS 2 → EliteX HyperOS 2 Enhancement**
- **HyperOS 3 → EliteX HyperOS 3 Enhancement**

---

## Step 2

- Magisk
- KernelSU
- KernelSU Next

---

## Step 3

Reboot your device.

# ⚠️ Xiaomi EU Specifications

The included **device_info.json** contains default specifications.

To display your own device specifications:

Navigate to

```text
/system/product/etc/
```

Open

```text
device_info.json
```

Replace the default values with your device specifications.

Example:

```json
{
  "basic": {
    "CPU": "Snapdragon 8 Elite",
    "Battery": "6000mAh",
    "Camera": "50MP + 50MP + 50MP",
    "Screen": "6.78-inch AMOLED",
    "Resolution": "3200 × 1440"
  },
  "camera": {
    "front_camera": "32MP",
    "rear_camera": "50MP + 50MP + 50MP"
  }
}
```

Save the file, recreate the ZIP while preserving the original folder structure, and then flash the modified module.

# ❓Frequently Asked Questions

### Which version should I install?

Install the version that matches your HyperOS version.

- HyperOS 2 → HyperOS 2 Module
- HyperOS 3 → HyperOS 3 Module

---

### Can I install the HyperOS 3 version on HyperOS 2?

No.

Each version is specifically built for its respective HyperOS framework.

---

### Does it work on MIUI?

No.

EliteX HyperOS Enhancement is designed exclusively for HyperOS.

---

### Which root solutions are supported?

- Magisk
- KernelSU
- KernelSU Next
